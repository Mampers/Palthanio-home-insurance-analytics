# Palthanio Home Insurance Analytics

## Overview

Palthanio Home Insurance Analytics is a portfolio project demonstrating how insurance data can be transformed into a modern analytics solution using SQL and Power BI.

The project simulates a home insurance company analysing profitability, claims performance, broker contributions, and risk exposure.

---

# Project Goals

This project demonstrates:

- dimensional modelling
- insurance KPI analysis
- Power BI reporting
- data engineering architecture
- business storytelling with analytics

---

# Architecture

The solution follows a **Medallion Architecture**.

Bronze → Silver → Gold

### Bronze
Raw data ingestion.

### Silver
Data cleaning and transformation.

### Gold
Analytics-ready dimensional model.
